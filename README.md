# image-annotator
Simple program to assign label to images for image classification. Labels are stored in a TSV file

How to use:
1. save images to be annotated in a folder named image_folder
2. run `python annotator.py image_folder`
3. in the GUI, select appropriate Gender and Status
4. click Next for next image
5. click save if you want to save your progress and stop annotating -> it will be saved in the tsv file
6. output will be saved in a tsv file in tsv directory
